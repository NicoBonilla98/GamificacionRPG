Deep, recessed text field — the look of the login vault. The field is an ink-well that lights its border gold on focus.

```jsx
<Input label="Nombre del héroe" placeholder="Ej. Thorne de Valdris" />
<Input label="Llave de la bóveda" type="password" placeholder="••••••••" />
```

Pass `hint` for a helper line. Forwards all native `<input>` props (`type`, `value`, `onChange`, `required`…).
