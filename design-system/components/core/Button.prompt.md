Action primitive — gold for the one decisive action on a screen, quieter outlines for everything else; never two gold buttons competing.

```jsx
<Button variant="primary" size="lg" fullWidth>Comenzar la aventura</Button>
<Button variant="complete" icon="✓">Completar</Button>
<Button variant="mini">Elegir</Button>
<Button variant="ghost">Simular cierre del día</Button>
```

Variants: `primary` (gold gradient CTA, glow), `complete` (gold outline, used on quest/chapter cards), `mini` (small gold outline for NPC/consumable pick), `ghost` (dashed muted, low-stakes), `danger` (rose outline). Sizes: `sm` · `md` · `lg`. Buttons run UPPERCASE and very heavy (950); ghost is the exception (sentence case, normal tracking).
