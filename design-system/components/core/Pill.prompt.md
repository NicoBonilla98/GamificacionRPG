Small uppercase chip for counts, tiers and status tags — pairs with section headings and rank rows.

```jsx
<Pill>3 pendientes</Pill>
<Pill tone="gold" icon="🗝">2 llaves</Pill>
<Pill tone="violet">Roster narrativo</Pill>
```

Tones: `neutral` (default panel chip), `gold`, `green`, `violet`, `danger`. Always uppercase, weight 900, pill-rounded.
